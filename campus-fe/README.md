## App in React

This is the Front End app connected with the backend built with NodeJS/Express + MongoDB + JWT. 


### Project setup

```
npm install
# or
yarn install
```

### Run

```
npm start
# or
yarn start
```

Open [http://localhost:8081](http://localhost:8081) to view it in the browser.
